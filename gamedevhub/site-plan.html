Overview

Purpose

This site will be to showcase my game ideas, information about them, and progress in completing them.

Audience

My audience is other people in the indie game dev community, and potential future employers.

Dynamic elements

dynamic elements will include game idea cards that are populated using Javascipt, and a projects page that has a search bar so users can easily search through ideas.
